GitHub Integration

When you click "Push to GitHub", the updated config.json file is committed directly to your repository. Use a Fine-grained Personal Access Token with Contents: Write permission on the repository.

1. Go to GitHub → Settings → Developer Settings → Personal Access Tokens → Fine-grained Tokens
2. Create a token → Select your portfolio repository → Enable Contents: Read and Write
3. Fill in GitHub Username, Repository, Branch and Token → Click "Push to GitHub" to save changes to config.json
